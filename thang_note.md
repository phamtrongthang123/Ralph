# Notes in English (for human discussion)
All code and external notes are LLM-generated and do not follow my personal style. This separate note exists so I can track things in my own words.

## Why this file exists

LLM-generated code is inherently hard to audit — it is syntactically correct and runs, but the internal logic can be wrong or do something unintended without anyone noticing. Having a separate human-written note to track progress and use the LLM for QA is therefore essential. This file is where I record what is happening in natural language, so I can quickly review without reading the code itself.
